# Node setup

## Install nvm

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash
```

Restart `bash` for the `nvm` command to be active.
Install latest node version, here latest version 14.

```bash
nvm install 14;
node -v;
```

## New project setup

```bash
mkdir project;
cd project;
git init;
npm init;
```
